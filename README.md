Lippupeli

Lippupelissä käyttäjän kuuluu arvata maan lippu joka näytetään kuvassa.
Käyttäjän kuuluu painaa rv:stä mielestään oikean maan nimeä ja painaa lopuksi submit.
Ohjelma ilmoittaa napin alapuolelle tekstillä oliko vastaus oikein vai väärin.

Oikein = vihreä teksti: "Oikein!"
Väärin = punainen teksti: "Väärin! Se oli (Oikea vastaus)."

Käyttäjä voi arvata useamman kerran arvaamalla ohjeiden mukaisesti.
